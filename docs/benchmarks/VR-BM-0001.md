# WallBench

Draft 0

## Abstract

WallBench is a fictional benchmark for evaluating coordination under semantic
pressure.

## 1. Purpose

WallBench measures whether a coordinated system can maintain a stable objective
while individual agents receive incomplete, delayed, or conflicting information.

## 2. Benchmark Families

### 2.1 Wall-Static

Agents receive fixed partial evidence.

The system must produce a coherent final state and preserve unresolved
uncertainty.

### 2.2 Wall-Drift

Agents receive sequential summaries of prior agent outputs.

The system must avoid semantic collapse.

### 2.3 Wall-Conflict

Agents receive evidence that supports incompatible interpretations.

The system must preserve disagreement without turning it into noise.

## 3. Metrics

`K`

Coordination efficiency.

`D`

Semantic drift.

`R`

Recovery after contradiction.

`P`

Provenance retention.

## 4. Reporting Rule

WallBench results are fictional unless backed by a runnable implementation and
clearly labeled as measured output.
