# Naming System

nyanya LLM names should be short, institutional, and durable.

## Prefixes

- `VR-TR` - technical report
- `VR-RFC` - request for comments
- `VR-BM` - benchmark
- `VR-RN` - research note
- `VR-REL` - release note

## Reserved Names

- Field
- Flow
- Merge
- WallBench
- Semantic Consensus Protocol

## Avoid

- long pseudo-academic compounds;
- fake affiliation with real labs;
- claims that sound validated when they are fictional;
- explaining the premise inside artifact titles.
